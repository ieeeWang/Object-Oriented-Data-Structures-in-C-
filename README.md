# Object-Oriented-Data-Structures-in-C++
Solutions of [Object-Oriented Data Structures in C++](https://www.coursera.org/learn/cs-fundamentals-1/home/welcome)


It is a nice (intermediate-level) open course to recall the way how C++ manage the memory, which is not needed anymore for using modern languages (e.g., python and Java).
The homeworks are done in a Linuxcompatible C++ work environment on (1) PC using the Windows Sub-system for Linux [WSL](https://docs.microsoft.com/en-us/windows/wsl/install-win10) and (2) Creating a Environment on [AWS Cloud9](https://aws.amazon.com/cloud9/).
